# MEV Streckenkunde – Starterpaket

## Schnellstart
1. Neues GitHub-Repo erstellen.
2. Inhalt dieses ZIP ins Repo hochladen.
3. Deine echte Karte als `assets/map.svg` ersetzen.
4. GitHub Pages aktivieren: Settings → Pages → Deploy from branch → main / root.

## Dateien
- `index.html` App (Wizard)
- `assets/mitarbeiter.json` Mitarbeiterliste
- `assets/stations_by_line.json` Linien/Stationen/Ortskenntnis
- `assets/index.json` Linienbeschreibungen (optional)
- `assets/map.svg` Platzhalter (ersetzen)
